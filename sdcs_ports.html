Symantec Data Center Security Manager and its agents communicate using specific ports for secure and efficient data exchange. Here's a breakdown:

Server Ports

Management Console:

Outbound communication to the management service: TCP ports 4443, 8006, and 8081.

Management Service:

Inbound communication from the management console: TCP ports 4443, 8006, and 8081.

Inbound communication with the UMC Web console: TCP port 8443.

Communication with agents: TCP port 443.

Outbound communication with SQL servers: TCP port 1433 (default) and UDP port 1434.

Agent Ports

Agent Communication:

Inbound communication from the management server: TCP port 2222.

Outbound communication to the management server: TCP port 443.
